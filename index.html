Linear Search

Linear search is a simple search algorithm that checks every element in a list one by one until it finds the target or reaches the end.

How It Works

  1. Start at the first element.
  2. Compare it with the target.
  3. If it matches, return the index.
  4. If not, move to the next element.
  5. Keep going until the target is found or the list ends.

Pseudocode

        

for each index in the list:
    if list[index] == target:
        return index

Python Example

        

def linear_search(arr, target):
    for i in range(len(arr)):
        if arr[i] == target:
            return i
    return -1

Algorithm Steps:

  1. Start from the first element (index 0).
  2. Compare the target value with the current element.
  3. If a match is found, return the index.
  4. If not, move to the next element.
  5. Repeat until the end of the list.
  6. If no match is found, return -1.
